### Heading Style
The presentation of can differ from the default styling of the rendered HTML heading using the `styleAs` prop.
<Preview>
  <div>
    <Heading as="h6" styleAs="h1">H6 styled as an H1</Heading>
    <Heading as="h5" styleAs="h2">H5 styled as an H2</Heading>
    <Heading as="h4" styleAs="h3">H4 styled as an H3</Heading>
    <Heading as="h3" styleAs="h4">H3 styled as an H4</Heading>
    <Heading as="h2" styleAs="h5">H2 styled as an H5</Heading>
    <Heading as="h1" styleAs="h6">H1 styled as an H6</Heading>
  </div>
</Preview>
