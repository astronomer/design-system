# Flash
An inline display to communicate a state to the user.

### Import
```js
import { Flash } from '@astronomer/spectra';
```

### Usage

#### Default (warning)
<Preview>
  <Story name="Default">
    <Flash>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Flash>
  </Story>
</Preview>

#### Flash Type
<Preview>
  <Story name="Type">
    <div>
      <Flash type="info">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Flash>
      <br />
      <Flash type="success">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Flash>
      <br />
      <Flash>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Flash>
      <br />
      <Flash type="error">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Flash>
    </div>
  </Story>
</Preview>

#### Dismissable
Adding `isDismissable` prop displays an "X" users can click to dismiss a Flash message. Add a `dismissKey` prop as well in order to persist the user's choice to dismiss in localStorage.
<Preview>
  <Story name="Is Dissmissable">
    <Flash isDismissable>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Flash>
  </Story>
</Preview>
